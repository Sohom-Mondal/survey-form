
# Survey Form

A simple survey form of a tech based company.


## Tech Stack

**Client:** React, TailwinCSS

**Host:** Github Pages


## Run Locally

Clone the project

```bash
  git clone https://github.com/Sohom-Mondal/survey-form.git
```

Go to the project directory

```bash
  cd survey-form
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
